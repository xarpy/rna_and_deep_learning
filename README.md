## ⚙️ Installation and Configuration

Follow the steps below to set up your development environment and run the project locally.

### 1.  Prerequisites

Make sure you have **Python 3** (version 3.14 or later) installed on your computer.

### 2. Virtual Environment Configuration

The project was created in a Jupyter notebook (`.ipynb`) and can be imported into Google Colab or run locally using any IDE of your choice. If you want to run it locally, it is recommended that you use a virtual environment (`venv`) to isolate the project’s dependencies from the operating system.

#### **Setting the Environment**

- Run the command below in the terminal, in the project's root directory:

```bash
python -m venv venv
```

- Then run the command below to install the recommended packages:

```bash
source <path-to-venv>/bin/activate
```

- After venv is activated, run this last command:

```bash
pip install -r requirements.txt
```

**Note**: We recommend setting up the Python environment in your specific IDE, along with the necessary configurations to run the **Jupyter** plugin, so that you can execute the notebook correctly.
